Quick Start:

The TestDataGenerator functionality was written to
easily generate test data, patterned after an instance
of an object.

As of v1.0.0.0 release, it currently supports DataTable,
but it is open to future structures.


Usage Example:

DataTable myRealDataTable = new DataTable();
